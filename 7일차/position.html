position : relative

콘텐츠에 상대좌표값을 부여하여 위치를 이동시킬 수 있음. 좌표값을 가져도 다른 콘텐츠에 영향을 주지 않음

상대좌표(left, right,top,bottom)

sticky

relative + fixed 조합 특징) 부모에 relative값을 주고 콘텐츠가 브라우저 상단에 위치할때 sticky fixed로 동작함 콘텐츠 하단이 상단에 위치할때 sticky 해제

position : absolute

절대좌표값은 부모좌표를 기준으로 이동시킬 수 있음. 다른 콘텐츠와 겹쳐짐. 부모는 절대좌표를 선언한 콘텐츠의 사이즈를 인식하지못함

절대좌표(left, right,top,bottom)

fixed

ablsolute와 동일한 값을 갖고 있음 특징) - 차이점은 스크롤 움직임에 영향을 받지않음 - 부모가 position값을 갖고 있어도 브라우저 기준으로 함
